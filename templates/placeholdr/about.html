Placeholdr is an application that was developed by 4 Computer Science students as part of a course project at the University of Glasgow.
The objective of our application is to allow people to browse through places of interest and create trips based on those places, with a community of people that provides ratings, comments, pictures.
Our application is open source and the project code can be viewed on Github.



Technologies used

  • django

    webserver middleware

  • Google Maps API

    accurate real-time information for mapping, navigation and places

  • Bootstrap

    CSS and JavaScript framework for beautiful website design

  • django-bootstrap4

    django application to integrate Boostrap

  • django-csp

    Content Security Policy for django

  • django-mathfilters

    a set of simple math filters for django

  • django-referrer-policy

    middleware class implementing the Referred-Policy header for django powered sites

  • django-geoposition

    django model field that can hold a geoposition, and corresponding widget

  • animate.css

    cross-browser library of CSS animations

  • wow.js

    Reveal CSS animation as you scroll down a page

  • pytz

    accurate and cross platform timezone calculations

  • Selenium

    web browser interaction automation from Python

  • GitHub

    web implementation of the git version control software

  • pythonanywhere

    small python-based website hosting
